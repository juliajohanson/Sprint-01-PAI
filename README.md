# Problema abordado
O problema abordado na sprint são as dificuldades do ChargeGrid intelligence que varia de problemas como controle de demanda, interoperapilidade de sistemas, gestão de gastos e gestões de terceiros.
# Proposta  
Nossa proposta é realizar um chatbot capaz de ser um assistente virtual tanto técnico como auxiliar geral na área do ChargeGrid intelligence da GoodWe.

# Tecnologias utilizadas  
Nós utilizamos o "olhama" que é um sistema de IA multimodal que irá responder as perguntas do usuário baseado no modelfile criado a partir de um "banco de dados" de perguntas inseridas para treina-lo.

# Justificativa técnica  
A justificativa sobre a criação do projeto foi a necessidade e utilidade de um assistente virtual especializado no ChargeGrid sendo capaz de auxiliar e informar novas pessoas sobre o projeto, tudo sendo capaz através do "olhama" que é um sistema de IA multimodal que apesar de ser "fraco" quando comparado aos outros exerce um desempenho exemplar no caso, juntamente do colab onde o integramos em um computador virtual, assim facilitando sua simulação.
